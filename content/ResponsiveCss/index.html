Responsive trong CSS

Responsive là gì?

Hiểu một cách đơn giản Web Responsive là phong cách thiết kế website làm sao cho phù hợp trên tất cả các thiết bị, mọi độ phân giải màn hình.


Nguyên tắc responsive

Responsive cho PC trước, sau đó sử dụng các công cụ như Dev Tools để responsive cho Tablet rồi đến Mobile.
Hướng đi: PC → Tablet → Mobile


Sử dụng responsive hiệu quả

Để sử dụng responsive hiệu quả bạn hãy thêm viewport vào file html của mình, xem hướng dẫn chi tiết Tại đây


Vì sao nên sử dụng em thay vì px trong media queries?

Việc sử dụng em

Responsive Cho Mobile

Responsive cho mobile: 0px → 739px
0 ≤ x < 740
739px = 46.1875em 
@media (max-width: 46.1875em)

Responsive Cho Tablet

Responsive cho mobile: 740px -> 1024px 
@media (min-width: 740px) and (max-width: 1023px)

Responsive Cho PC

Responsive cho mobile: 1024px -> Infinity 
@media (min-width: 1024px)

Responsive Chi Tiết

Chi tiết cho nhiều dòng máy và cấu hình máy khác nhau
