Vì sao nên dùng Reset CSS:

Bởi vì mỗi trình duyệt đều sẽ có css mặc định khác nhau, việc sử dụng CSS Reset sẽ giúp đưa mọi thứ về 0, giúp cho việc hiển thị trên mọi trình duyệt đều giống như nhau.
